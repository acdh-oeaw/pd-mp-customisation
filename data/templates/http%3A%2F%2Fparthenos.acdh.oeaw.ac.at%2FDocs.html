Documentation

These pages inform about the functionality of the application and various aspects of the dataset

Application

PARTHENOS Discovery is a customisation of the semantic knowledge management platform Metaphactory. It operates solely on top of a SPARQL endpoint and allows end users to visualize and explore the data in a user-friendly, highly customisable manner. PARTHENOS Discovery is considered complementary to the Joint Resource Registry, in that both operate on - and give access to - the same dataspace.

Further information: "metaphactory: A Platform for Knowledge Graph Management" http://www.semantic-web-journal.net/system/files/swj1950.pdf

Read about the main features of the application: A comprehensive description about the customisations of Metaphactory and considerations/findings about the tools for resource discovery in general can be found in the deliverable PARTHENOS D6.4 Report on services and tools (final). There is also a comprehensive documentation of the functionality Metaphactory provides.

Dataset

PARTHENOS Discovery gives access to the metadata records, contributed by its participating partners, which have been converted into a common semantic model, the PARTHENOS Entities Model.

Under Statistics you can get a broad overview of the overall dataset: Which sources contribute what information? How many instances are available per class? How often are certain properties used? etc.
