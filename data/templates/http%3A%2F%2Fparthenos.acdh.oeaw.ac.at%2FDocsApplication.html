Application

PARTHENOS Discovery is a customisation of the semantic knowledge management platform Metaphactory. It operates solely on top of a SPARQL endpoint and allows end users to visualize and explore the data in a user-friendly, highly customisable manner. PARTHENOS Discovery is considered complementary to the Joint Resource Registry, in that both operate on - and give access to - the same dataspace. However while JRR provides a minimal canonic record for the PARTHENOS entities. PARTHENOS Discovery gives access to all entities, properties and relations of the overall dataset and allows for richer exploration through interlinking of the entities and advanced query capabilities.

Further information: "metaphactory: A Platform for Knowledge Graph Management" http://www.semantic-web-journal.net/system/files/swj1950.pdf

Read about the main features of the application: A comprehensive description about the customisations of Metaphactory and considerations/findings about the tools for resource discovery in general can be found in the deliverable PARTHENOS D6.4 Report on services and tools (final). There is also a comprehensive documentation of the functionality Metaphactory provides.
