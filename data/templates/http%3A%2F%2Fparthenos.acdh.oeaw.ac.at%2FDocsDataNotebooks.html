Data Notebooks

Being worked on. See PD User Texts gdoc.

These pages provide an example of how to explore the PARTHENOS dataset programmatically through Jupyter notebooks.

Data notebooks are increasingly popular and very effective way to develop and run code through your browser. You don't need to install anything locally, you can combine code and text to explain and document, contributing to radically simplified sharing and reproducibility (your colleague can completely "re-play" your notebook).
