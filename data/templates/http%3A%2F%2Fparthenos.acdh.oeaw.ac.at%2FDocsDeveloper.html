Developer Documentation

Being worked on. See PD User Texts gdoc.

These page provides information for Developers.

PARTHENOS Discovery is technically a customisation of the Metaphactory knowledge management platform. Metaphactory provides a customisation mechanism through custom-tailored pages in which predefined components combined with parameterized SPARQL-queries can be used to display data at hand. The set of available components is described in Metaphactory help pages.

All these custom template pages that make up PARTHENOS Discovery are made available in a dedicated GitHub repository: https://github.com/acdh-oeaw/pd-mp-customisation.
