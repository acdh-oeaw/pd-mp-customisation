Data Notebooks

This page informs about possibilities to explore the PARTHENOS dataset programmatically through Jupyter notebooks using python programming language.

The data notebooks are increasingly popular and very effective way to develop and run code through your browser. Therefore they are ideal means for manifold data science tasks especially for working in a collaborative distributed environment. They allow radically simplified sharing and reproducibility (your colleague can completely "re-play" your notebook) You do need a dedicated [jupyter](https://jupyter.org/) environment, however next to simple options to setup one locally, there are more and more offers providing appropriate environment as a service.

In the dedicated GitHub repository acdh-oeaw/pd-labs there is more information on the use of Jupyter noteboooks, as well as a sample Jupyter notebook (Querying PD.ipynb) demonstrating possible programmatic interaction with the PARTHENOS dataspace.
