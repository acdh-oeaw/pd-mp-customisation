Developer Documentation

This page provides technical background information about the application.

PARTHENOS Discovery is technically a customisation of the Metaphactory knowledge management platform (Version 3.1). Metaphactory allows to explore, navigate through and manage large knowledge graphs, represented as RDF. It operates on top of any SPARQL 1.1 compliant triplestore. Metaphactory provides a customisation mechanism through custom-tailored application and template pages in which predefined components combined with parameterized SPARQL-queries can be used to craft custom views on the data at hand. The set of available components is described in Metaphactory help pages.

All these custom pages that make up PARTHENOS Discovery are made available in a dedicated GitHub repository: https://github.com/acdh-oeaw/pd-mp-customisation.

The underlying triplestore is an instance of blazegraph, being updated with new data once a week (every Saturday night). The data is provided as RDF dump by projet partner CNR, who is running the infrastructure for large-scale aggregation and transformation d-net. The dumps are available under https://virtuoso.parthenos.d4science.org/dump/. The ingestion into blazegraph takes around 11 hours for 1.46 billion triples (as of 2019-09-22) in Bulk Data Load mode.
